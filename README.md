Netflix Smart Discovery & Analytics Platform

Overview

An interactive web application that combines Netflix content analytics with personalized content discovery.

Users can filter Netflix content by genre, country, and content type while exploring analytics on Netflix’s catalog.

Features

* Personalized content recommendations
* Genre-based filtering
* Country filtering
* Movie and TV Show filtering
* Netflix catalog analytics
* Interactive Streamlit dashboard

Technologies Used

* Python
* Pandas
* Streamlit
* Data Cleaning
* Exploratory Data Analysis (EDA)

Dataset

Netflix Titles Dataset containing 8,800+ movies and TV shows.

Analytics Included

* Movies vs TV Shows
* Top Ratings
* Top Countries
* Top Genres
